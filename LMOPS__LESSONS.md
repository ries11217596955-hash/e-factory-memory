LMOPS LESSONS

MATURE LESSONS
1. Local LLM is a memory / analysis engine, not project truth source.
2. API access and GUI access are different things and must not be confused.
3. Remote tunnel facts are volatile and must not be treated as stable architectural truth.
4. Project memory should not depend on temporary remote-access details to remain understandable.
5. Use LM as helper / analyzer, not as authoritative source of verified facts.
